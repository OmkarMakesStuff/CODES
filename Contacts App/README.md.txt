## How to Use

When you run the script, you'll be presented with a menu offering the following options:

1. **Add a new contact**:  
   This option prompts you to enter a name and phone number. If the contact already exists, it will notify you.

2. **View all contacts**:  
   This option lists all contacts stored in the `contacts.txt` file, showing each contact's name and phone number.

3. **Delete a contact**:  
   This option allows you to delete a contact by entering the contact's name. If the contact exists, it will be removed from the file.

4. **Search a contact**:  
   This option prompts you to enter a name, and if that contact exists, it displays their phone number.

5. **Exit**:  
   Exits the program.

### Example Interaction:

```bash
===== Contact Book =====
1. Add a new contact
2. View all contacts
3. Delete a contact
4. Search a contact
5. Exit
Enter your choice (1-5): 1
Enter name: John Doe
Enter phone number: 123-456-7890
Contact for John Doe added successfully!
